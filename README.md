# Tic Tac Toe Game (AI Based)

This is a simple Tic Tac Toe game built using Python. The game can be played either between two players or against the computer.

In single player mode, the computer has three difficulty levels. In easy mode, it plays random moves. In medium mode, it tries to block the player and win when possible. In hard mode, it uses the minimax algorithm, which makes it very difficult to beat.

This project helped me understand how game logic works and how artificial intelligence can be used to make decisions in games. It also improved my understanding of problem solving and recursion.

Features:

* Single player and two player modes
* Three difficulty levels (easy, medium, hard)
* AI-based decision making using minimax algorithm
* Input validation and proper game flow

How to run:

1. Make sure Python is installed
2. Run the file in terminal or VS Code:
   python filename.py

The game will start in the terminal and guide you through the steps.
